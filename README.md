# Natural Language Processing Practicals

This repository contains the practical implementations for the Natural Language Processing (NLP) course. Each directory includes the source code and necessary datasets for the respective assignments.

## List of Practicals

1. **Practical 1: Tokenization**
   - Implementation of various tokenization techniques (Whitespace, WordPunct, Treebank, Tweet, and MWE Tokenizers).
   - Comparative analysis of Stemming and Lemmatization.

2. **Practical 2: Word2Vec**
   - Text representation using the Word2Vec model.
   - Exploratory data analysis on the provided dataset.

3. **Practical 3: Text Cleaning and Preprocessing**
   - Advanced text preprocessing and cleaning techniques.
   - Vectorization using TF-IDF and storage of processed models.

## Requirements
- Python 3.x
- NLTK
- Pandas
- Scikit-learn
- Gensim
